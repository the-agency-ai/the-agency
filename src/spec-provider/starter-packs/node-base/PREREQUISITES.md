# Prerequisites

## Required

- [ ] macOS, Linux, or WSL2 (Windows)
- [ ] Terminal access
- [ ] Git installed (`git --version`)
- [ ] GitHub account with CLI configured (`gh auth status`)

## Recommended

- [ ] VS Code or similar editor with TypeScript support
- [ ] Claude Code installed (`claude --version`)

## Check Your Environment

```bash
# Should all return versions
git --version
node --version  # May not exist yet - that's OK
gh auth status  # Should show logged in
```
