# Supabase Auth Starter Pack

**Authentication with Supabase**

## What This Pack Does

Sets up authentication with:

- Supabase project creation (via CLI)
- Auth configuration
- Protected routes middleware
- Login/signup components

## Why Supabase?

| Choice         | Alternative  | Why We Chose This                       |
| -------------- | ------------ | --------------------------------------- |
| Supabase       | Auth0, Clerk | Free tier, PostgreSQL, self-host option |
| Email/password | Social only  | Universal, no OAuth setup               |
| Middleware     | HOC          | App Router native                       |

## Dependencies

**Required:** `node-base`, `react-app`
