# Release D46.R1 (v46.1) — AgencyV3 Foundation

**Project branding:** AgencyV3 — the installer threshold.
**Symbolic tag:** `agency-v3` (co-located with `v46.1` on the same commit)
**Versioning scheme:** D.R (Day . Release-number) — today = Day 46, first release = R1.

## TL;DR

v46.1 is a **structural reset release**. Framework code directory renamed from `claude/` to `agency/`; subdir organization (`REFERENCE/`, `README/`); workstream consolidation; cruft removal (docs/, integrations/, assets/, receipts/); content-level reference sweep. **Agency Release Versioning** moves to D.R format (`46.1` is today's first release). The `agency-v3` symbolic tag marks arrival of the installer/updater foundation.

**Note:** v46.1 does NOT include the full build-boundary infrastructure (Python build tool + src/claude/src/agency split + dual-tracked build output). That's deferred to a follow-up release. v46.1 ships the structural foundation; subsequent releases layer in build + distribution automation.

## Migration runbook (adopter: v45.2 → v46.1)

### Prep
```bash
# Pre-migration verification
./agency/tools/agency-verify v46
```

Exit codes:
- 0: clean
- 10: tree-shape mismatch (claude/ present, agency/ absent)
- 11: settings.json stale
- 12: agent registration stale
- 13: ISCP smoke fail
- 14: hook ENOENT

### Update (path rewriting for adopters)
Since v46.1 does NOT yet have the full `agency update --migrate` path (deferred to v46.2 with build tool), manual steps for adopters:

1. Rename local framework dir: `git mv claude agency`
2. Create REFERENCE/ and README/ subdirs: `mkdir agency/REFERENCE agency/README && git mv agency/REFERENCE-*.md agency/REFERENCE/ && git mv agency/README-{ENFORCEMENT,RECEIPT-INFRASTRUCTURE,SAFE-TOOLS}.md agency/README/`
3. Move data/ to config/: `git mv agency/data/* agency/config/ && rmdir agency/data`
4. Update `.claude/settings.json` hook paths: `sed -i '' 's|claude/hooks|agency/hooks|g; s|claude/tools|agency/tools|g' .claude/settings.json`
5. Update `@import` in agent registrations: `sed -i '' 's|@claude/|@agency/|g' .claude/agents/**/*.md`
6. Update skill `required_reading:`: `find .claude/skills -name 'SKILL.md' -exec sed -i '' 's|claude/|agency/|g' {} +`
7. Verify: `./agency/tools/agency-verify-v46 --customer` — expect exit 0
8. Bump local manifest: update `agency/config/manifest.json` `agency_version` to `46.1`
