# The abstraction ladder

From switches to AI:

1. Hardware switches (early computers)
2. Assembly
3. High-level languages
4. Frameworks and libraries
5. **AI-augmented development** — we are here

Each step abstracts away mechanical toil. Each step was criticized as "not real programming" at the time.

# The Enforcement Triangle

Every capability has three parts:

| Layer | What | Why |
|---|---|---|
| **Tool** | Does the work | Permissions |
| **Skill** | Tells the agent when to use it | Discovery |
| **Hookify rule** | Blocks the raw alternative | Compliance |

Build all three. Not one, not two.
