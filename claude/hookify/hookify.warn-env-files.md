  - field: file_path
    operator: regex_match
    pattern: \.env$

**Editing a `.env` file directly.**

This project uses Doppler for environment variable management. `.env.example` files exist as documentation only. `.env.local` files are auto-generated by `prototype:up`. Do not create or edit `.env` files directly.
