# React App Starter Pack

**Next.js 15 with App Router**

## What This Pack Does

Sets up a React web application with:

- Next.js 15 App Router
- TypeScript (extended from node-base)
- Tailwind CSS + shadcn/ui
- Basic layout and routing

## Why This Stack?

| Choice     | Alternative  | Why We Chose This           |
| ---------- | ------------ | --------------------------- |
| Next.js 15 | Vite, CRA    | Full-stack, App Router, RSC |
| App Router | Pages Router | Future-proof, RSC, layouts  |
| Tailwind   | CSS Modules  | Utility-first, fast         |
| shadcn/ui  | MUI, Chakra  | Composable, owns components |

## Dependencies

**Required:** `node-base`
