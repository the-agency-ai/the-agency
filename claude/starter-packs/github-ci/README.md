# GitHub CI Starter Pack

**Continuous Integration with GitHub Actions**

## What This Pack Does

Sets up CI/CD with:

- GitHub Actions workflow
- Build verification
- Lint/type checking
- Test running

## Why GitHub Actions?

| Choice         | Alternative       | Why We Chose This                 |
| -------------- | ----------------- | --------------------------------- |
| GitHub Actions | CircleCI, Jenkins | Integrated, free for public repos |
| pnpm           | npm               | Matches local development         |
| Node 22        | Earlier versions  | LTS, consistent                   |

## Dependencies

**Required:** `node-base`
