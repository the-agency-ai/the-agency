# PostHog Analytics Starter Pack

**Product Analytics with PostHog**

## What This Pack Does

Sets up product analytics with:

- PostHog project creation (via CLI/API)
- Client-side event tracking
- Server-side event capture
- Feature flags integration
- Session recording (optional)

## Why PostHog?

| Choice  | Alternative         | Why We Chose This                                 |
| ------- | ------------------- | ------------------------------------------------- |
| PostHog | Mixpanel, Amplitude | Open-source, self-host option, generous free tier |
| Cloud   | Self-hosted         | Simpler setup, managed infrastructure             |
| JS SDK  | REST API only       | Auto page views, session recording                |

## Dependencies

**Required:** `node-base`, `react-app`
