# {{AGENT_NAME}} Onboarding

## Quick Start

1. Launch the agent:
   ```bash
   ./claude/tools/myclaude {{WORKSTREAM}} {{AGENT_NAME}}
   ```

2. Review current work:
   - Check `WORKLOG.md` for sprint progress
   - Check `ADHOC-WORKLOG.md` for ad-hoc tasks

3. Read agent identity:
   - Review `agent.md` for purpose and responsibilities

## Key Resources

- Agent directory: `claude/agents/{{AGENT_NAME}}/`
- Workstream: `claude/workstreams/{{WORKSTREAM}}/`
- Knowledge: `KNOWLEDGE.md` in this directory

## Getting Help

```bash
./claude/tools/collaborate housekeeping housekeeping "I need help with..."
```
