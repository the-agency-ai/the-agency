# markdown-pal Onboarding

## Quick Start

1. Launch the agent:
   ```bash
   ./tools/myclaude markdown-pal markdown-pal
   ```

2. Review current work:
   - Check `WORKLOG.md` for sprint progress
   - Check `ADHOC-WORKLOG.md` for ad-hoc tasks

3. Read agent identity:
   - Review `agent.md` for purpose and responsibilities

## Key Resources

- Agent directory: `claude/agents/markdown-pal/`
- Workstream: `claude/workstreams/markdown-pal/`
- Knowledge: `KNOWLEDGE.md` in this directory

## Getting Help

```bash
./tools/collaborate housekeeping housekeeping "I need help with..."
```
